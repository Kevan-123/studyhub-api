# StudyHub API

A Django REST Framework backend API to help users manage their **study goals**, **tasks**, and **learning resources** efficiently. Built as part of the ALX Backend Web Development Capstone Project.

---

## Features

- JWT Authentication (login & token refresh)
- CRUD operations for **Study Goals**
- CRUD operations for **Tasks**
- CRUD operations for **Categories** and **Learning Resources**
- User-specific data filtering — users can only see their own goals, tasks, and resources
- RESTful API endpoints for easy integration with frontend applications
